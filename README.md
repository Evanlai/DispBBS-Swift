主要功能為：  
  
不需登入即可  
．瀏覽本日熱門文章
．瀏覽各大看板的文章  
．搜尋看板、瀏覽最近看過的看板  
  
註冊 Disp BBS 的帳號登入後  
．可以在看板發表文章  
．編輯已發表的文章  
．在看板發文回覆其他人的文章  
．查看站內信箱，可寄信與回覆信件  
．在站內信箱新增備忘錄  
．編輯文章時可上傳圖片
